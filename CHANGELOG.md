- Runtime v3

**Migration**

```json

  "builds": [
    {
      "src": "index.php",
      "use": "now-php"
    }
  ]
}
```

➡️

```json
{
  "functions": {
    "api/*.php": {
      "runtime": "now-php@0.0.7"
    }
  },
  // Optionally provide routes
  "routes": [
    { "src": "/(.*)",  "dest": "/api/index.php" }
  ]
}
```
